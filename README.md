# openapi-php-sdk-status

> This repository is companion repository to verify the status of the Openapi® PHP SDK.

## Overview

This repository contains scripts and tools to check the status of the Openapi® PHP SDK. It helps developers ensure that the SDK is up-to-date, functioning correctly, and compatible with the latest Openapi specifications.
